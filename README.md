# 📘 Place-Mate – Centralized Placement Preparation & Progress Tracker

Place-Mate is a full-stack web application designed to simplify and enhance the **student placement preparation workflow**.  
It consolidates drive tracking, resource management, goal tracking, authentication, and progress analytics into a single unified platform.

---

## 🚀 Features

### 🔐 Authentication System
- Secure **Login / Signup**  
- Password validation with error handling  
- Protected routes & session management  

### 📊 Personalized Dashboard
- Overview of tasks, drives, and preparation progress  
- Dynamic cards/widgets  
- User-specific recommendations  

### 📝 Placement Drive Manager
- Add/update company drives  
- Track application status  
- Bookmark important drives  

### 🎯 Roadmap + Goal Checklist
- Track daily/weekly preparation milestones  
- Add custom goals  
- Progress visualization indicators  

### 📚 Resource Storage & Notes Manager
- Save links, attachments, notes  
- Organize by topic or category  
- Easy access from dashboard  

### 💻 Coding Practice Logger
- Record coding questions solved  
- Track platform, difficulty, topic  
- Preparation analytics  

### ⚡ App Performance & State Management
- **Redux Toolkit / Context API** for global state  
- Route-based code splitting  
- Component-level memoization
